1-апта: JavaScript негіздері

HTML & CSS білетіндер үшін JavaScript-ке кіріспе

JavaScript дегеніміз не?

JavaScript — веб-беттерді интерактивті етуге арналған бағдарламалау тілі. HTML бетінің құрылымын өзгертпей, динамикалық өзгерістер жасауға мүмкіндік береді.

Мысалы, батырманы басқанда мәтінді өзгерту, калькулятор құру немесе API арқылы деректерді алу үшін JS қолданылады.

Мысал:

<!DOCTYPE html>
<html>
<body>

<h1 id="greeting">Сәлем!</h1>
<button onclick="changeText()">Басуды сынап көр</button>

<script>
function changeText() {
  document.getElementById("greeting").innerHTML = "Сәлем, JavaScript!";
}
</script>

</body>
</html>

Тапсырмалар:

1. Жай alert хабарламасын шығарыңыз.
2. console.log арқылы өз атыңызды шығарыңыз.
3. Батырма басқанда парақтағы мәтінді өзгертіңіз.
4. HTML парағында екі батырма жасап, әрқайсысы әртүрлі мәтін шығаратындай код жазыңыз.

Айнымалылар (Variables)

Айнымалылар — деректерді уақытша сақтау үшін қолданылады. JavaScript-де үш негізгі түрі бар: var, let, const.

Мысалдар:

// var мысалы
var name = "Ayan";
console.log(name);

// let мысалы
let age = 25;
age = 26; // қайта тағайындауға болады

// const мысалы
const pi = 3.14;
// pi = 3.1415; // қатеге әкеледі

Тапсырмалар:

5. Өз атыңызды let арқылы айнымалыға сақтап, консольге шығарыңыз.
6. const арқылы туған күніңізді сақтаңыз, оны қайта тағайындауға әрекет жасаңыз.
7. Айнымалыға сан мен жол мәнін тағайындап, оларды қосып көріңіз.
8. Қолданушыдан prompt арқылы жасын сұрап, 18-ден үлкен болса хабарлама шығарыңыз.

Мәлімет түрлері (Data Types)

JavaScript-де негізгі мәлімет түрлері: Number, String, Boolean, Null, Undefined, Symbol, Object.

Мысалдар:

let name = "Ayan"; // String
let age = 25;       // Number
let isStudent = true; // Boolean
let middleName = null; // Null
let lastName;          // Undefined

Тапсырмалар:

9. 3 әртүрлі мәлімет түрін айнымалыда сақтап, консольге шығарыңыз.
10. null және undefined айырмашылығын көрсететін код жазыңыз.
11. Өз атыңыз, жасыңыз, хоббиіңізді объект ретінде сақтаңыз және консольге шығарыңыз.

Түрлерді ауыстыру (Type Conversion)

JavaScript-де мәндерді бір типтен екінші типке ауыстыруға болады: String, Number, Boolean. Бұл кейде автоматты (implicit) түрде де жүреді.

Мысалдар:

let num = 42;
let str = String(num); // "42"
let bool = Boolean(str); // true

let text = "123";
let number = Number(text); // 123

Тапсырмалар:

12. Санды жолға, жолды санға айналдырыңыз.
13. Boolean түрге бірнеше мәндерді ауыстырып, нәтижелерін бақылаңыз.
14. Prompt арқылы сан сұрап, оны жолға, содан кейін керісінше түрге ауыстырып, нәтижесін көрсетіңіз.

Шаблон жолдар (Template Literals)

ES6-де `${}` көмегімен жолдарды динамикалық құрастыруға болады.

Мысал:

let name = "Ayan";
let age = 25;
console.log(`Менің атым ${name} және мен ${age} жастамын.`);

Тапсырмалар:

15. Өз атыңыз бен сүйікті түстеріңізді шаблон жол арқылы шығарыңыз.
16. Қолданушыдан prompt арқылы атымын сұрап, шаблон жолмен хабар шығарыңыз.
17. Бірнеше айнымалыны шаблон жолдар арқылы бір сөйлемге біріктіріңіз.

Debugging & Console

Кодты тексеру және қателерді табу үшін console.log(), console.warn(), console.error() қолданылады.

Мысал:

let x = 5;
console.log("x мәні:", x);
console.warn("Бұл ескерту");
console.error("Бұл қате");

Тапсырмалар:

18. Айнымалыға сан беріп, оны консольге шығарыңыз.
19. Қате мәнін intentionally шығарып, console.error() арқылы тексеріңіз.
20. Кодқа бірнеше console.log қосып, айнымалы мәндерін бақылаңыз, содан кейін қателікті табыңыз.

Мини жобa: Жеке ақпарат картасы

Осы аптада үйренгендерді біріктіріп, "Жеке ақпарат картасы" жасаңыз: аты, жасы және сүйікті түсі. Қолданушыдан ақпарат сұрап, оны экранға шығарыңыз.

Мысал жоспары:

let name = prompt("Атыңызды енгізіңіз:");
let age = prompt("Жасыңызды енгізіңіз:");
let color = prompt("Сүйікті түсті енгізіңіз:");

console.log(`Сәлем, ${name}! Сіз ${age} жастасыз. Сүйікті түсіңіз: ${color}`);

Тапсырмалар:

21. Қолданушыдан бірнеше мән сұрап, оларды консольге шығарыңыз.
22. Егер қолданушы жасын 18-ден төмен енгізсе, ескерту шығарыңыз.
23. Түрін ауыстырып, жасын сан ретінде салыстырыңыз.
